# Device Utilities

## isAndroid

Returns true if the current OS is Android

### Usage

```jsx
import { DeviceUtils } from 'hc-mobile-app-ui'

const padding = DeviceUtils.isAndroid() ? 10 : 20;
```

## isIos

Returns true if the current OS is iOS

### Usage

```jsx
import { DeviceUtils } from 'hc-mobile-app-ui'

const padding = DeviceUtils.isIos() ? 20 : 10;
```
