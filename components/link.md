### Props

| Name        | Type     | Description                                                   |
| ----------- | -------- | ------------------------------------------------------------- |
| label       | string   | Label for the link.                                           |
| onLinkPress | function | Function to be called when the link is pressed.               |
| to          | string   | Path to be redirected to. It can be a URL or an in-app route. |
